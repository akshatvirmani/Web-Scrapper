## To run the program

``` {
  "type": "module",
  "name": "web-scrapping",
  "version": "1.0.0",

  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
  "license": "ISC",
  "dependencies": {
    "puppeteer": "^22.6.5"
  }
}
```
